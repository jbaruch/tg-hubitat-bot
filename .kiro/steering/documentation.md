# Documentation Guidelines

## Context7 for Documentation

Always use Context7 for language, libraries, and APIs documentation. Context7 provides up-to-date, accurate documentation for:

- Programming languages (Kotlin, Java, Python, etc.)
- Libraries and frameworks (Kotest, Ktor, Mockito, etc.)
- APIs and SDKs

When you need documentation about a language feature, library, or API:
1. Use Context7 to fetch the latest documentation
2. Reference the documentation in your implementation
3. Ensure code follows the documented best practices

This ensures that all code is written according to current standards and best practices.
